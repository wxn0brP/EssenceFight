Keyboard Controls

Type a command, then press Enter to execute. Esc clears buffer.

d{num}{zone}{slot} Deploy card from hand
a{fromPos}{toPos} Attack
ee End turn
nn Next phase
e{zone}{slot}.{effectIdx}[target] Use effect (zone: d=deck, g/c/r=board)
ArrowUp/Down Navigate history
h Toggle help

Zones: g = ground, c = castle, r = runes, d = deck

d12g1 -> 13th card -> ground-1

ag1c0 -> ground-1 attacks castle-0

ed1.1 -> deck card 1, effect 1

eg0.2g1 -> ground-0 effect 2, target ground-1

Press h or Esc to close
